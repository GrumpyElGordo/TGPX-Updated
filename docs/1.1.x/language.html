Introduction

The TGPX language file allows you to translate error and other informative messages into your native language. This document describes how to modify the language file settings.

Control Panel Editing Interface

All changes to your language settings will be made through the control panel. To edit your language settings access the Templates > Language File function from the control panel menu. A page will load where you can edit all of the available language settings. Each of the items should be self-explanatory by reading the existing text. You can modify the settings to change the messages or translate them into your language. Once you have made the changes you want, press the Save Language File button at the top of the page.

Value Placeholders

Some of the default messages have value placeholders (%d, %s, etc) embedded in them so that dynamic information can be inserted into them. Each of the language file settings that has these placeholder values are described below.
    CURLE_COULDNT_CONNECT %s will be replaced by a software generated error message
    CURLE_HTTP_RETURNED_ERROR %s will be replaced by a the returned HTTP status
    INVALID_URL %s will be replaced by the name of the improperly formatted URL
    DESCRIPTION_LENGTH The first %d will be the minimum description length
    The second %d will be the maximum description length
    MAXIMUM_KEYWORDS %d will be replaced by the maximum keywords value you specified in the General Settings interface
    ACCOUNT_EXPIRED The first %s will be the starting date of the account
    The second %s will be the ending date of the account
    CATEGORY_FULL %s will be replaced by the category name
    BLACKLISTED The first %s will be the blacklist item that was matched
    BLACKLISTED_REASON The first %s will be the blacklist item that was matched
    The second %s will be the reason why this item is in the blacklist
    BROKEN_URL The first %s is the type of URL submitted
    The second %s is the error message associated with this URL
    EXCESSIVE_LINKS %d is the maximum number of links you allow, taken from the general settings interface
    INVALID_FORMAT %s is the format (pictures or movies) allowed in the associated category
    BAD_THUMB_COUNT The first %d is the minimum number of thumbs allowed
    The second %s is the maximum number of thumbs allowed
    USERNAME_LENGTH The first %d will be the minimum username length
    The second %d will be the maximum username length
    PASSWORD_LENGTH The first %d will be the minimum password length
    The second %d will be the maximum password length
    INVALID_IMAGE_SIZE The first %d will be allowed image width
    The second %d will be the allowed image height
    PREVIEW_DOWNLOAD_FAILED %s will be an error message indicating why the download failed
    IMAGE_DOWNLOAD_FAILED %s will be an error message indicating why the download failed
    SLOW_DOWNLOAD The first %f will be the download speed received from the server
    The second %d will be the minimum allowed download speed
    SMALL_CONTENT %f will contain the minimum allowed content size
    EXCESSIVE_CATEGORIES %d is the maximum number of categories you are allowing to be submitted
