E-mail Templates

Table of Contents
Introduction

The e-mail templates are the templates that control the content of the e-mail messages that will be sent to your users when their galleries are processed, if they need to reset their partner account password, etc. This document will give you an overview of the variables available in each of the e-mail templates.

Editing E-mail Templates

To edit the e-mail templates access the Templates > E-mail Messages function from the control panel menu. This will display a page where you can select which template you want to edit. Once you have selected the template you want to edit, press the Load Template button. You will be returned to the same page, but there will now be three fields that you will need to fill in:
    Subject The full subject of the e-mail message
    Text Body The text body of the e-mail message. This message will be displayed for users that have text-only e-mail readers.
    HTML Body The HTML body of the e-mail message. This message will be displayed for users that have HTML capable e-mail readers.
    TIP: Click on the icon to automatically convert the text body to HTML format.
When you are done making changes to the template, press the Save Template button to save your changes.

Templates and Their Variables

This section will describe all of the available templates and the variables you can use within each template.
    email-admin-reports.tpl

    This e-mail message is sent to administrator accounts when there are cheat/broken gallery link reports waiting to be processed

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$reports_waiting} - The number of cheat reports waiting to be processed


    email-admin-requests.tpl

    This e-mail message is sent to administrator accounts when there are partner account requests waiting to be processed

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$requests_waiting} - The number of partner account requests waiting to be processed


    email-admin-scanner.tpl

    This e-mail message is sent to administrator accounts when the gallery scanner has finished running

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$scanned} - The number of galleries that were scanned
      {$total} - The total number of galleries that should have been scanned


    email-gallery-approved.tpl

    This e-mail message is sent to a gallery submitter when their gallery is approved

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$gallery} - The gallery data; see the Common Variables section for details


    email-gallery-confirm.tpl

    This e-mail message is sent to a gallery submitter when their submission must be confirmed by e-mail

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$gallery} - The gallery data; see the Common Variables section for details


    email-partner-added.tpl

    This e-mail message is sent to a partner when their account request is approved

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$partner} - The partner account data; see the Common Variables section for details


    email-partner-expiring.tpl

    This e-mail message is sent to a partner when their account is about to expire

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$partner} - The partner account data; see the Common Variables section for details


    email-partner-reset-confirm.tpl

    This e-mail message is sent to a partner when they request that their password be reset

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$partner} - The partner account data; see the Common Variables section for details
      {$confirm_id} - The confirmation ID that must be present in the confirmation URL


    email-partner-reset-confirmed.tpl

    This e-mail message is sent to a partner containing their username and password when they reset their password

      {$config} - The TGPX configuration settings; see the Common Variables section for details
      {$partner} - The partner account data; see the Common Variables section for details
