Categories

Table of Contents
Introduction

Categories are one of the basic building blocks for your TGP and allow submitters and administrators to specify what category(ies) each gallery fits into. Before you can begin accepting gallery submissions you will need to setup at least one category. This document describes how to manage your categories.

Viewing/Searching Categories

To view or search your currently configured categories, access the Settings > Manage Categories function from the control panel menu. This will display a page where you will be able to see the currently configured categories and search them if you wish.

In the table that displays the categories, you will find a couple of functions that you can run on a single category. Each of those functions are described below.
    Click this icon to edit the category. A window will appear where you will be able to edit all of it's settings.
    Click this icon to delete this category.
In addition to running these functions on single categories, you can also run the delete function on multiple categories at once. To run the delete function, put a check in the checkbox next to the categories you want to delete, then press the Delete button at the bottom of the page.

The search results will display a few different pieces of information about the gallery. Each column of the search results is described below.
    Name The name of the category
    Pictures This category allows picture galleries
    This category does not allow picture galleries
    This category allows thumbnail previews for picture galleries
    Movies This category allows movie galleries
    This category does not allow movie galleries
    This category allows thumbnail previews for movie galleries
    Hidden This category is hidden and cannot be selected by gallery submitters
    This category is not hidden and can be selected by gallery submitters
    Last Submit The date and time that the last gallery was submitted to this category through the submit.php script
    Sorter The value corresponding to the "Sort:" field that you selected for the current search
Adding a Category

To add a new category access the Settings > Manage Categories function from the control panel menu. This will display your currently configured categories. Near the top of the page in the header bar that reads "Manage Categories" click on the icon. This will display a popup window where you will be able to enter the data for the new category that you want to add. Each of the form fields are described below.
    General Settings
    Category Name(s) The name(s) you want to use for the category(ies). You can enter one category per line to add multiple categories at once.
    Meta Description The value to display in the TGP page's meta description tag. This value can be displayed on any page that is assigned to this category. See the meta_categories.txt example template included with the software.
    Meta Keywords The value to display in the TGP page's meta keywords tag. This value can be displayed on any page that is assigned to this category. See the meta_categories.txt example template included with the software.
    Submissions Per Day The maximum number of galleries that can be submitted to this category each day
    Make this category hidden
    If this is checked this category will not appear on the end-user gallery submission form and users will not be able to submit galleries to this category
    Picture Gallery Settings
    Allow picture galleries in this category
    Check this box to allow picture gallery submissions in this category
    File Extensions Enter a comma separated list of file extensions allowed for pictures
    Minimum Thumbs The minimum number of thumbnails that must appear on the gallery
    Maximum Thumbs The maximum number of thumbnails that may appear on the gallery
    Minimum Filesize The minimum allowed filesize of the pictures on the gallery
    Allow preview thumbnails for this format
    Check this box to allow submitters to provide preview thumbnails with their gallery submissions for this category
    Preview Dimensions The size of preview thumbnails allowed in this category. Select Custom --> to enter a new size in WxH format or select one of the existing sizes from the drop down list.
    Preview Annotation The annotation to display on the preview thumbnails for galleries in this category. This will only appear if you have defined one or more annotations.
    Movie Gallery Settings
    Allow movie galleries in this category
    Check this box to allow movie gallery submissions in this category
    File Extensions Enter a comma separated list of file extensions allowed for movies
    Minimum Thumbs The minimum number of thumbnails that must appear on the gallery
    Maximum Thumbs The maximum number of thumbnails that may appear on the gallery
    Minimum Filesize The minimum allowed filesize of the movies on the gallery
    Allow preview thumbnails for this format
    Check this box to allow submitters to provide preview thumbnails with their gallery submissions for this category
    Preview Dimensions The size of preview thumbnails allowed in this category. Select Custom --> to enter a new size in WxH format or select one of the existing sizes from the drop down list.
    Preview Annotation The annotation to display on the preview thumbnails for galleries in this category. This will only appear if you have defined one or more annotations.
Once you have filled in all of the fields as you want them, press the Add Category button. This will add the new category to the database, and it will be activated and ready for galleries immediately.

The Default Category

Any time you add a new category you will notice that many of the form fields have been pre-filled for you. This data is taken from the default category settings, which can be edited. To edit the default category, access the Settings > Manage Categories function from the control panel menu. This will display your currently configured categories. Near the top of the page in the header bar that reads "Manage Categories" click on the icon. This will display a popup window where you will be able to edit the data for the default category.

The default category settings are the same as when you add a new category, with th exception that you do not enter a category name. The form fields described above have the same meaning when editing the default category as they do when adding/editing a defined category. Once you've got the settings configured how you want them, press the Save Default Settings button to record this information.

The default category is also used when you are importing galleries and you tell the import function to create a new category for any that are imported with the galleries but don't already exist. The default category settings will be used for the new category(ies) that are created during the gallery import process.

Editing a Category

To edit a category you will first need to locate it through the View/Search interface. When you locate the category you want to edit, click on the icon to bring up the category editing interface. This interface is nearly identical to the category addition interface, and has the same form fields with the same properties. You can make the changes that you want, and then press the Update Category button to save your changes.

Deleting a Category

Deleting a category will also delete all of the galleries contained in that category!

To delete a category you will first need to locate it through the View/Search interface. When you locate the category you want to delete, click on the icon, which will ask you to confirm the deletion.

You can also delete several categories at once by putting a check in the box next to the categories you want to delete, then pressing the Delete button at the bottom of the page.
