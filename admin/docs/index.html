General Information

This documentation covers the TGPX software functions, and tries to be as thorough as possible. If there is something you cannot find in this manual, please check the knowledge base first. If you still cannot find an answer to your question, please use the support forum for assistance. If you do find any errors or inconsistencies in the documentation, please let me know so it can be corrected.

If you have just had TGPX installed for the first time, please see the First Steps section of this manual first. It will help get you started using TGPX and will guide you through the first things you need to do.
